# 战斗基础流程 v1

> 创建时间: 2026-06-27

## 概述

竖屏 PixiJS 游戏，720×1440。初始主角向上移动，敌人从上方生成向下靠近，自动攻击。

## 游戏规则

### 单位

- 我方初始有 1 个单位（主角），后续可通过三选一获得更多
- 每个单位有独立血量
- 我方所有单位死亡 → 失败

### 战斗

- 敌人进入攻击范围 → 自动发射飞弹（单体命中）
- 敌人有远程单位（发射单体飞弹）和近战单位（靠近后攻击）
- 杀敌积累经验

### 波次

- 共 20 波敌人
- 每波敌人数量递增（比上一波更多）
- 波间有短暂间隔
- 杀光一波 → 刚好升级 → 触发三选一

### 三选一

- 杀光一波后出现三选一
- 选项：新单位 or 对当前单位的强化
- 选择后继续下一波
